# Image Viewer

## Description

This is a simple static web application that displays a single image. It's designed to be clean, responsive, and easy to deploy using GitHub Pages.

## Features

- **Image Display:** Shows a single image on the page.
- **Responsive Design:** Adapts to different screen sizes for optimal viewing on desktops and mobile devices.
- **Modern Design:** Uses a clean and visually appealing design with a modern color scheme.
- **Self-Contained:** Includes all necessary HTML, CSS, and JavaScript within a single file.
- **Easy Deployment:** Can be easily deployed using GitHub Pages.

## How to Use

1.  Clone the repository to your local machine.
2.  Open the `index.html` file in your web browser.
3.  Alternatively, deploy the project to GitHub Pages to view it online.

## Technologies Used

-   HTML
-   CSS (Embedded)
-   JavaScript (Vanilla - Embedded)
